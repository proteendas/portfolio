# PROTEEN.DEV — Personal Portfolio

A comic-book styled, high-performance personal portfolio website for **Proteen Das** — Fullstack Developer & Frontend Specialist based in Bengaluru, India.

## Features

- Animated loader with progress bar
- 3D hero text with mouse-tilt parallax
- Custom cursor with magnetic hover effects
- Scroll-velocity skew + card parallax via GSAP ScrollTrigger
- Dark mode toggle
- Animated marquee banners
- Comic-panel project & experience cards with 3D tilt
- Scroll progress HUD
- Text scramble animation
- Fully responsive + reduced-motion support

## Tech Stack

- **HTML / CSS / Vanilla JS** — zero frameworks, zero build tools
- **GSAP 3** + **ScrollTrigger** — all animations
- **Google Fonts** — Bangers, Archivo, JetBrains Mono

## Project Structure

```
proteen-portfolio/
├── index.html   # Markup & content
├── styles.css   # All styles & theming
└── script.js    # GSAP animations, cursor, loader, interactions
```

## Running Locally

Just open `index.html` in a browser — no build step required.

```bash
open index.html
# or serve with any static server
npx serve .
```
